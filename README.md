# Form Actions Demo

This is a React application demonstrating form actions using React 19 and Vite.

## Features

- User signup form with validation
- Header component
- Built with modern React hooks and form handling

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd react-form-actions-demo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To run the development server:

```bash
npm run dev
```

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```

## Project Structure

- `src/App.jsx`: Main application component
- `src/components/Header.jsx`: Header component
- `src/components/Signup.jsx`: Signup form component
- `src/util/validation.js`: Validation utilities

## Technologies Used

- React 19
- Vite
- ESLint for code linting
